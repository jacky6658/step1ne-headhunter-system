# 📋 履歷分析教學指南

## 目的
當獵頭顧問提供履歷給 AI 時，需要進行**系統化分析**，以快速評估候選人的適配度、優劣勢，並推薦合適的職缺。

---

## 分析框架 (5 層次)

### 第一層：基本信息提取 (基礎事實)

提取以下信息：
- **姓名** / **聯絡方式**（電話、Email）
- **目前居住地** / **國籍/工作權**
- **目前職位** / **目前公司**（如有）

**檢查清單：**
- ✅ 聯絡方式完整且有效
- ✅ 確認工作權（特別是海外候選人）
- ✅ 判斷地點是否符合職缺要求

**範例（AVA CHEN）：**
```
姓名：AVA CHEN (雷家乙)
電話：(+1) 778-926-4272
Email：thisbigsister@gmail.com
目前地點：加拿大（從區號判斷）
⚠️ 特殊情況：海外居住，需確認來台工作意願
```

---

### 第二層：技能與經驗評估 (核心能力)

#### 2.1 軟體工具 & 技術技能

- **列出所有提及的工具/系統**
  - MS Office (Word, Excel, PowerPoint)
  - Google Workspace (Trello, Slack)
  - 專業系統（如 Sage 50、BIM、PLC 等）

- **評估深度**：
  - 🟢 **精通**：日常使用、能培訓他人
  - 🟡 **熟悉**：能獨立操作、解決基本問題
  - 🔴 **基礎**：使用過但不熟練

**範例（AVA CHEN）：**
```
軟體工具：
• MS Office → 🟡 熟悉（工作中使用）
• Google Workspace → 🟡 熟悉（Trello, Slack）
• Sage 50 → 🟡 熟悉（財務報表）
• BIM / AI 工具 → 🔴 無經驗

技術技能評估：**基礎文書類** ⭐⭐（不適合技術職）
```

#### 2.2 語言能力

- **列出所有語言**
- **識別認證**（如 CELPIP、TOEFL 等）
- **評估等級**：
  - 🟢 **認證 + 工作證明**
  - 🟡 **工作經驗證明**
  - 🔴 **自述**

**範例（AVA CHEN）：**
```
語言能力：
• 中文（母語）
• 英文（CELPIP 加拿大英檢認證 ✅）
• 雙語優勢 → 適合跨國企業

評估：⭐⭐⭐⭐⭐ (認證 + 工作環境)
```

#### 2.3 核心專業技能

- **識別該職務的關鍵技能**
- **評估掌握深度**
- **識別缺口**

**範例（AVA CHEN）：**
```
核心技能清單：
✅ 文件管控 / 行政支援（Sage 50 系統）
✅ 客戶溝通（飯店主管經驗）
✅ 協調能力（總經理助理）
❌ 招募專業（如要 HR 崗位）
❌ 技術能力（AI、工程類）

分類：
- 行政管理：⭐⭐⭐⭐
- 技術能力：⭐⭐
```

### 第三層：工作經歷分析 (穩定度 & 深度)

#### 3.1 按時間順序整理

```
公司名稱 | 職位 | 起迄 | 月數 | 主要職責
---------|------|------|------|--------
A 公司 | 主管 | 2023/11-2026/02 | 15M | 文件管控、客戶服務
B 公司 | 助理 | 2023/03-2023/10 | 7M | 資料輸入
```

#### 3.2 穩定度評分

**計算：**
- **轉職次數**：越少越好（3年內變動不超過 2 次）
- **平均任職時間**：計算每份工作的平均月數
- **最近 Gap**：上一份工作結束到現在的間隔

**評估邏輯：**
```
穩定度 = (平均任職月數 / 預期) × (1 - 轉職頻率) × (最近Gap評分)

示例計算（AVA CHEN）：
- 轉職 2 次（在可接受範圍）
- 平均任職：11 個月（偏短）
- 最近 Gap：無（直接從上份工作結束）
- 評估：⭐⭐⭐ (中等穩定)
```

#### 3.3 離職原因分析

**判斷方式：**
- 📜 **履歷上寫的**（主動說明）→ 參考性高
- 🔍 **從職位轉換判斷**（Data Entry → Guest Supervisor）→ 升遷或興趣轉換
- ❓ **不明原因**（沒有解釋）→ 面試時詢問

**範例（AVA CHEN）：**
```
SRS Windows (7個月) → Quality Inn (15個月)
判斷：從資料輸入升級到飯店主管
→ 可能是主動尋求發展機會（正面訊號 ✅）
```

### 第四層：與職缺匹配度評估

#### 4.1 建立匹配矩陣

對於**每個可能的職缺**，評估：

```
職缺：HR 招募專員 (人資部，50k-70k)

| 維度 | JD 要求 | 候選人 | 符合度 | 備註 |
|------|---------|--------|--------|------|
| 文書管控 | 必須 | ✅ 強 | 100% | Sage 50 經驗 |
| 行政支援 | 必須 | ✅ 強 | 100% | 總經理助理 |
| 溝通協調 | 必須 | ✅ 強 | 90% | 飯店主管經驗 |
| 招募經驗 | 加分 | ❌ 無 | 0% | 需培訓 |
| 英文能力 | 加分 | ✅ 認證 | 100% | CELPIP |
| 年資要求 | 1-3年 | ✅ 符合 | 95% | 有 2.8 年 |

總體匹配度：85% (適配但需培訓)
```

#### 4.2 優先順序評估

**高度適配（80%+）** → 優先推薦
```
✅ HR 招募專員 - 85%
  理由：行政專長 + 溝通能力符合，文件管控經驗直接應用
  缺點：無招募經驗，需 2-4 周 onboarding
```

**中度適配（60-79%）** → 次選
```
⚠️ 律師文件管理師 - 72%
  理由：文件管控背景符合，英文能力強
  缺點：需要法律知識，完全陌生領域
```

**不適配（<60%）** → 不推薦
```
❌ AI 工程師 - 15%
❌ BIM 工程師 - 10%
  原因：完全缺乏技術背景
```

### 第五層：綜合評估 & 報告

#### 5.1 優勢分析 (3-5 項)

列出候選人的**關鍵優勢**，以及為什麼這對職缺有幫助：

```
✅ 優勢 1：雙語能力 + 認證
   → 適合跨國企業、客戶溝通職位

✅ 優勢 2：文件管控系統經驗
   → HR、法務、行政類職位直接應用

✅ 優勢 3：客戶服務主管經驗
   → 溝通協調能力強，適合團隊互動

✅ 優勢 4：加拿大工作經驗
   → 有海外適應能力、英文實戰能力
```

#### 5.2 劣勢分析 (2-4 項)

列出**需要改進或培訓的點**：

```
⚠️ 劣勢 1：技術技能有限
   → 不適合 AI、工程、技術類職位
   → 適合行政、HR、文書類

⚠️ 劣勢 2：職涯跳動
   → 從資料輸入跳到飯店主管
   → 可能反映興趣廣泛或不夠專精
   → 面試時詢問職涯規劃

⚠️ 劣勢 3：當前在加拿大
   → 需確認來台工作意願
   → 可能涉及簽證、搬遷等
```

#### 5.3 適合職缺推薦 (優先排序)

```
🟢 第一優選（推薦度 85%+）
   職缺：HR 招募專員
   薪資：50k-70k
   理由：文件管控 + 行政經驗直接應用，溝通能力強

🟡 第二選擇（70-84%）
   職缺：行政專員 / 律師文件管理師
   薪資：55k-75k
   理由：文書能力符合，但需培訓領域知識

🔴 不推薦
   AI 工程師、BIM 工程師、技術職位
   理由：完全缺乏技術背景
```

### 第六層：分級評分表

建立**標準化評分表**，便於快速決策：

```

-----------------|------|-----------|-----

工作經驗質量      | 5分  | 3 分     | ⭐⭐⭐
技術能力          | 5分  | 2 分     | ⭐⭐
語言能力          | 5分  | 5 分     | ⭐⭐⭐⭐⭐
職涯穩定度        | 5分  | 3 分     | ⭐⭐⭐
溝通協調能力      | 5分  | 4 分     | ⭐⭐⭐⭐
文化適配度        | 5分  | 4 分     | ⭐⭐⭐⭐
成長潛力          | 5分  | 4 分     | ⭐⭐⭐⭐
-----------------|------|-----------|-----
總計              | 40分 | 29 分    | 總級別：B

評級標準：
35-40 分 → A 級 (優秀候選人，優先推薦)
28-34 分 → B 級 (良好候選人，中度推薦)
21-27 分 → C 級 (可考慮，需對應特定職缺)
< 21 分  → D 級 (不推薦)
```

## 常見問題

### Q1：如何判斷「短期工作」是負面信號？
**A：** 
- < 3 個月 → 通常是負面訊號（需詢問原因）
- 3-6 個月 → 中性（可能試用期不符合預期）
- 6-12 個月 → 可接受（試用期 + 適應期）
- 12+ 個月 → 正面訊號

如果有 2 次以上短期工作 → 詢問職涯規劃是否清晰

### Q2：海外候選人的評估重點？
**A：**
- ✅ 確認工作權（簽證、居住權）
- ✅ 語言認證 (CELPIP、TOEFL 等)
- ✅ 文化適應能力
- ✅ 搬遷意願 & 成本
- ⚠️ 可能的簽證延遲

### Q3：如何評估「職涯轉換」？
**A：**
- 📍 同一產業內轉換 → 正面（累積經驗）
- 📍 不同產業轉換 → 中立（需詢問動機）
- 📍 來回轉換 → 負面（職涯規劃不清）
- 💡 詢問：為什麼轉換？有什麼可轉移的技能？

### Q4：沒有寫出的技能怎麼評估？
**A：**
- 🔍 從工作職責推斷（如「財務報表」→ Excel 能力）
- 📞 在面試時確認
- 🎓 從教育背景推斷
- ⚠️ 不要過度解讀

## 進階技巧

### 技巧 1：尋找「隱藏訊號」

```
表面：資料輸入員 → 飯店主管
隱藏訊號：
✅ 自我驅動（主動升遷）
✅ 可塑性強（不同領域都適應）
✅ 溝通能力（升遷到主管層級）
⚠️ 專精度可能有限
```

### 技巧 2：「技能轉移」評估

```
舊職位的技能 → 新職位的應用

HR 招募專員需要：文件管控、溝通、組織能力
AVA 的背景：
✅ 文件管控（Sage 50）
✅ 溝通能力（飯店主管）
✅ 組織能力（行政支援）
→ 技能 70% 轉移，需 20% 培訓招募專業
```

### 技巧 3：面試重點準備

基於履歷分析，準備 3-5 個核心面試問題：

```
1. 【職涯規劃】
   「為什麼從資料輸入轉到飯店主管？未來職涯目標是什麼？」
   
2. 【加拿大經驗】
   「在加拿大的工作經驗如何幫助你適應這個職位？」
   
3. 【技能應用】
   「你使用 Sage 50 的經驗中，最複雜的任務是什麼？」
   
4. 【來台意願】
   「從加拿大來台灣工作，有什麼顧慮嗎？」
   
5. 【行業知識】
   「對招募專業/HR 領域的了解程度？有興趣學習嗎？」
```

## 最佳實踐

1. **不要過度解讀**
   - 只分析履歷中明確寫出的內容
   - 不確定的部分用「可能」或「需確認」標記

2. **保持客觀**
   - 避免個人偏見
   - 用數據與事實支撐評價

3. **考慮企業文化**
   - 不只看技能，也看文化適配度
   - 例：穩定度 vs 創新精神

4. **預留成長空間**
   - 評估候選人的學習潛力
   - 90% 完美 vs 70% 但成長快 → 後者可能更好

5. **溝通清晰**
   - 用表格、視覺化呈現
   - 推薦與理由要明確
   - 避免專業術語堆砌
