## 人選卡片總覽

人選卡片有 **5 個 Tab**：

| Tab | 用途 | 常用度 |
|-----|------|--------|
| 📋 基本資訊 | 管理人選所有資料、匯入履歷、編輯資訊 | ⭐⭐⭐⭐⭐ |
| 📅 進度追蹤 | 記錄聯繫/面試/Offer 進度 | ⭐⭐⭐⭐ |
| 📝 備註紀錄 | 日常筆記和溝通紀錄 | ⭐⭐⭐ |
| 🤖 AI 總結 | 一鍵複製提示詞，讓 AI 做全面人才分析 | ⭐⭐⭐⭐⭐ |
| ✨ AI 匹配結語 | 一鍵複製提示詞，讓 AI 做職缺匹配深度分析 | ⭐⭐⭐⭐⭐ |

## 基本資訊 Tab

### 負責顧問指派
- 點「指派給我」快速指派
- 點「更改」可搜尋並指派給其他顧問
- ⚠️ 轉派時，新顧問會收到**站內通知 + Telegram 推送**

### 目標職缺綁定
- 點「指定」選擇目標職缺
- 綁定後，AI 匹配結語 Tab 和電話篩選腳本才能正常運作

### 編輯人選資料
1. 點右上角「✏️ 編輯」進入編輯模式
2. 修改需要的欄位
3. 點「儲存」完成

**可編輯欄位：**
- 基本：姓名、職位、地點、年資、電話、Email、技能、學歷
- 個人：出生年月日（自動算年齡）、性別、英文名
- 求職：產業、語言、證照、薪資、到職時間、管理經驗
- 動機：求職狀態、轉職原因、主要動機、不接受條件、競爭 Offer
- 顧問：顧問備註（⚠️ 非常重要，AI 分析時會優先參考）、關係程度

### 電話篩選腳本
1. 先綁定「目標職缺」
2. 點「📞 生成電話篩選腳本」
3. AI 會根據人選資料 + 目標職缺自動生成結構化的電話篩選流程
4. 點「複製」把腳本貼到你需要的地方

### 外部連結
- LinkedIn URL、GitHub URL、作品集 — 點「編輯」可新增或修改
- Google Drive 履歷連結 — 直接可點開查看

## 匯入 PDF 履歷

### 操作步驟

1. 點人選卡片右上角的「📄 匯入履歷」
2. 選擇履歷格式：
   - **自動偵測**（推薦）— 系統自動判斷格式
   - **104 人力銀行** — 從 104 下載的履歷
   - **LinkedIn** — LinkedIn 匯出的 PDF
   - **通用格式** — 其他格式的履歷
3. 拖拉 PDF 檔案到上傳區或點擊選擇檔案
4. 系統會自動解析並列出所有欄位
5. 勾選你要匯入的欄位（預設全選有值的）
6. 點「套用選取欄位」

### 重要提醒

- ✅ 「個人簡介」欄位會自動存到**自傳（biography）**區塊
- ✅ 匯入後原有資料會被覆蓋（被勾選的欄位）
- ✅ 可以取消勾選不想覆蓋的欄位

## 深度資訊

> 💡 **自傳、作品集、語音評估**是人選的「深度資訊」。AI 分析時會特別參考這些資料，有這些資訊的人選會得到更精準的評估和更高的分數。

### 自傳（Biography）

**位置：** 資訊 Tab 中間區域，橘色漸層卡片

**如何新增/編輯：**
1. 找到橘色的「📝 自傳 / 自我介紹」區塊
2. 點「+ 新增自傳」或「✏️ 編輯」
3. 輸入候選人的自我介紹
4. 點「💾 儲存」

**自傳來源：**
- 匯入 PDF 履歷時，「個人簡介」欄位會自動填入
- 候選人直接提供的自我介紹
- 面談後顧問整理的候選人自述

**AI 怎麼用自傳：**
- 提取**職涯動機**和**自我定位**
- 分析**軟實力信號**（表達力、邏輯性、主動性）
- 判斷與目標職缺的**文化適配度**
- 識別**紅旗信號**（抱怨前雇主、目標不切實際等）

### 作品集（Portfolio URL）

**位置：** 外部連結區域

**如何新增：**
1. 找到外部連結區域
2. 點作品集旁的「編輯」
3. 貼上作品集連結（GitHub Pages、Behance、個人網站等）

### 語音/面談評估

**位置：** 資訊 Tab 下方

**如何新增：**
1. 找到「語音評估」區塊
2. 點「+ 新增」
3. 填入音檔 URL（可選）和分析結果
4. 點「儲存評估」

## AI 總結 Tab

> 🤖 **這是你最強大的工具之一！** AI 會幫你做全面的人才分析、職缺匹配推薦、能力評估和行動建議。

### 什麼時候用？
- ✅ 收到新人選時，快速了解人選全貌
- ✅ 不確定人選適合什麼職缺時
- ✅ 需要向主管報告人選狀況時
- ✅ 準備面試前想了解人選優劣勢

### 操作步驟

1. 打開人選卡片 → 點「🤖 AI 總結」Tab
2. 看一下上方自動生成的**一句話定位**和**關鍵資訊**
3. 點下方的「**📋 複製 AI 分析提示詞**」按鈕
4. 把複製的提示詞貼到你的 AI（ChatGPT / Claude / 本地 AI）
5. AI 會：
   - 自動呼叫 API 取得系統職缺
   - 分析人選的所有資料（含自傳、作品集、語音評估）
   - 產出 7 大分析項目
   - **自動呼叫 API 把分析結果寫回系統**（`ai_summary` 欄位）

### AI 會產出什麼？

| # | 項目 | 說明 |
|---|------|------|
| 1️⃣ | 一句話定位 | 20 字內精準描述核心價值 |
| 2️⃣ | Top 3 最匹配職缺 | 從系統職缺中挑選最適合的，含匹配分數 |
| 3️⃣ | 能力分析 | 核心優勢 + 能力缺口 |
| 4️⃣ | 自傳與深度洞察 | 從自傳/作品集/面談中提取的洞察 |
| 5️⃣ | 風險評估 | 穩定性、薪資、到職時間等風險 |
| 6️⃣ | 建議顧問提問 | 5 個面試問題（技能/行為/動機/風險）|
| 7️⃣ | 顧問行動建議 | 下一步做什麼 |

### AI 回寫後的效果

AI 分析完成後會自動把結果寫回系統。下次打開這個人選的 AI 總結 Tab，你會看到頂部出現一個**綠色的「🤖 AI 總結分析結果」卡片**，包含：
- 📌 一句話定位
- 🎯 Top 匹配職缺（含分數）
- ✅ 核心優勢
- ⚠️ 風險/待確認
- 🔍 深度洞察（如有自傳/作品集）
- 💡 行動建議

## AI 匹配結語 Tab

> ✨ **當你已經有明確的目標職缺時用這個！** 比 AI 總結更深入，專注在一個職缺的匹配分析。

### 什麼時候用？
- ✅ 確定人選要推某個職缺時
- ✅ 要寫推薦報告給客戶時
- ✅ 準備向用人主管 Pitch 這位候選人時
- ✅ 需要詳細的匹配度評分和推薦話術時

### 操作步驟

1. **先綁定目標職缺**（在基本資訊 Tab）
2. 打開人選卡片 → 點「✨ AI 匹配結語」Tab
3. 系統會自動顯示 **Top 5 系統職缺匹配**（含分數和匹配/缺少技能）
4. 拉到最下方，點「**📋 複製 AI 匹配分析提示詞**」按鈕
5. 把複製的提示詞貼到你的 AI
6. AI 會：
   - 自動呼叫 API 取得目標職缺詳細資料
   - 做 8 大匹配分析
   - **自動呼叫 API 把匹配結果寫回 `ai_match_result`**

### AI 會產出什麼？

| # | 項目 | 說明 |
|---|------|------|
| 1️⃣ | 匹配度總評 | 0-100 分 + 各維度評分 + 推薦等級 |
| 2️⃣ | 核心優勢 | 3-5 項相對於職缺的競爭力 |
| 3️⃣ | 風險與劣勢 | 硬傷 vs 可補足 |
| 4️⃣ | 自傳與深度洞察 | 從深度資訊中的發現 |
| 5️⃣ | 薪資分析 | 期望 vs 預算 + 談判策略 |
| 6️⃣ | 建議顧問提問 | 5-8 個面試問題 |
| 7️⃣ | 推薦話術 | 3-5 句向用人主管的推薦詞 |
| 8️⃣ | 顧問行動建議 | 具體下一步 |

### AI 回寫後的效果

結果寫回後，人選卡片的**匹配結語 Tab** 和**列表頁**都會顯示：
- 匹配分數（0-100）
- 推薦等級（強力推薦/推薦/觀望/不推薦）
- 匹配/缺少技能標籤
- 顧問建議

## 進度追蹤 Tab

### 快速新增進度

1. 點進度追蹤 Tab
2. 點擊對應的進度按鈕（聯繫階段/面試階段/Offer 等）
3. 選擇「快速原因」或自行填寫備註
4. 點「確認新增」

### 常用進度流程

```
未開始 → AI推薦 → 聯繫階段 → 面試階段 → Offer → on board
                                                    ↘ 婉拒
```

### 快速原因範例

| 階段 | 常用原因 |
|------|---------|
| 聯繫階段 | 已加入 LinkedIn、已寄 InMail、已電話聯繫 |
| 面試階段 | 已安排面試、第一輪完成、技術測試已送出 |
| Offer | 客戶已發 Offer、薪資協商中、候選人考慮中 |
| 婉拒 | 候選人婉拒（薪資/地點/興趣）、客戶拒絕 |

## 備註紀錄 Tab

- 用於記錄日常溝通重點、面試回饋、客戶反應等
- 每筆備註自動加上時間戳和你的名字
- 支持刪除過時的備註

> 💡 **重要提醒**：跟人選聊過後的重點，建議寫在「顧問備註」欄位（編輯模式裡），而不是這裡的備註 Tab。因為 AI 分析時會**優先參考顧問備註**。

## 每日工作流程建議

### 🌅 早上：處理新進人選

```
1. 打開「今日新增」人選清單
2. 對每個新人選：
   a. 匯入 PDF 履歷（如有）→ 自動填入自傳
   b. 綁定目標職缺
   c. 切到 AI 總結 Tab → 複製提示詞 → 貼給 AI 分析
   d. AI 自動回寫結果到系統
   e. 根據 AI 建議決定下一步
```

### 🌞 下午：推進現有人選

```
1. 查看「聯繫階段」和「面試階段」的人選
2. 對要推薦的人選：
   a. 切到 AI 匹配結語 Tab → 複製提示詞 → 貼給 AI 分析
   b. AI 自動回寫匹配結果
   c. 用 AI 產出的「推薦話術」向客戶推薦
   d. 記錄進度（進度追蹤 Tab）
```

### 🌙 收尾：記錄和回顧

```
1. 更新今天聯繫過的人選的「顧問備註」
2. 補充語音面談評估（如有電話面談）
3. 檢查是否有需要跟進的人選
```

## 提升 AI 分析品質的秘訣

1. **填寫自傳**：有自傳的人選，AI 分析會更精準，推薦更到位
2. **填寫顧問備註**：你跟人選聊過的第一手資訊，AI 優先參考
3. **綁定目標職缺**：AI 匹配結語的品質完全取決於職缺資訊的完整度
4. **做語音面談評估**：讓 AI 知道你對人選的觀察
5. **完善工作經歷**：這是 AI 分析最重要的資料來源

## 常見問題

**Q: AI 分析結果寫回系統後在哪裡看？**
A:
- AI 總結 → 打開 AI 總結 Tab，頂部綠色卡片
- AI 匹配結語 → 打開 AI 匹配結語 Tab，看到分數和推薦等級

**Q: 可以重新跑 AI 分析嗎？**
A: 可以！再複製一次提示詞貼給 AI，新的結果會覆蓋舊的。

**Q: AI 提示詞包含哪些人選資料？**
A: 所有已填的資料都會被包含：基本資訊、技能、工作經歷、穩定度、自傳、作品集、語音評估、顧問備註等。

**Q: 沒有自傳/作品集的人選，AI 會扣分嗎？**
A: 不會。沒有深度資訊的人選，AI 給中間分數（不加分也不扣分）。有深度資訊的人選可以加分拉開差距。

**Q: 人選轉派給別的顧問，他會收到通知嗎？**
A: 會！系統會發站內通知，如果有設定 Telegram Bot 也會推送到 TG。
